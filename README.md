A tiny CLI utility to print your public IP address.

### Install

```sh
$ npm install -g getmyip
```

### Run

```sh
$ getmyip
```

##### Available options:
```sh
-v, --verbose  Show more details
-h, --help     output usage information
-V, --version  output the version number
```
